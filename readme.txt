The official plugin for adding widgets and services provided by İzlesene to your WordPress-powered website.

== Description ==

Registers İzlesene as an oEmbed provider so pasting an İzlesene video URL will turn it automatically into an embed player.

More features are coming soon.

== Installation ==

1. Install İzlesene for WordPress either via the WordPress.org plugin directory, or by uploading the files to your server (in the /wp-content/plugins/ directory).
1. Pasting İzlesene video URLs should now be replaced
